# Hockey Pong — Squad Team

## Project Context

**Project:** Hockey Pong — 2D pixel art multiplayer hockey pong game
**Stack:** HTML5 Canvas, JavaScript/TypeScript, Node.js, WebSockets
**Description:** A classic Hockey Pong game with pixel art graphics. Single-player mode with AI opponent, and 2-player online multiplayer via shareable invite links. Players choose 1P or 2P mode; in 2P mode, a link is generated for the opponent to join the session.
**User:** Jose Corral
